# DNA Analysis Toolkit

## What is this?

A Python-based bioinformatics tool that analyzes DNA sequences from FASTA files.

---

## Features

- Reads FASTA files
- Calculates nucleotide counts (A, T, G, C)
- Computes GC content
- Generates RNA sequences
- Produces protein translation (ORF)
- Generates GC content visualization

---

## How to Run

Open terminal in project folder and run:

```bash
python main.py --file data/sample.fasta
